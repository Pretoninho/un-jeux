# Pitch & traduction — comment expliquer le jeu

> But de ce fichier : figer **comment parler du jeu** à quelqu'un qui ne trade pas, sans
> trahir la profondeur visée pour la niche. Deux niveaux de discours assumés (memo §12) :
> **dehors** = genre + métaphore, zéro jargon ; **dedans** = vrai vocabulaire trader, qui
> devient une *récompense de profondeur*, pas une barrière d'entrée.

---

## Tagline (une phrase, zéro jargon)

> **« Enrichis-toi tant que la fête dure — une chute arrive, tu ne sais pas quand, sors avant les autres. »**

## Le genre, pour un gamer : *press-your-luck* (pousse ta chance)

Comme tenter une salle de plus dans un roguelike avant de mourir, ou le all-in au poker.
La règle émotionnelle est universelle, aucun mot de finance requis :

> **L'avidité paie… jusqu'au moment où elle te ruine.**

## La métaphore sans finance : les chaises musicales

- La musique joue = tout monte, tout le monde gagne.
- Plus ça dure, plus c'est tendu : une **jauge de danger invisible** se remplit.
- La musique s'arrête d'un coup → ruée sur trop peu de chaises = le krach.
- **Ton skill** : lire des indices peu fiables (dans le brouillard) pour quitter la piste
  **avant** la coupure.

C'est littéralement le moteur (fragilité cachée + signaux bruités + contagion), raconté
à un enfant.

## La fantaisie à vendre (le « pourquoi j'y joue »)

Pas « simuler la finance ». C'est :

> **« Être le malin qui voit le crash venir et sort pile au bon moment — pendant que les
> autres se font ratatiner. »**

Schadenfreude + tension + sang-froid. Ça se vend à n'importe qui.

## Table de traduction (trader → gamer)

À garder en tête pour improviser une explication face à un non-initié.

| Terme trader | Traduction gamer |
| --- | --- |
| Fragilité `F` (cachée) | une **jauge de catastrophe** qui se remplit, invisible |
| Levier | un **multiplicateur de mise** : ×3 les gains **et** la mort |
| Krach / crise | le **wipe** (tout le monde meurt en même temps) |
| Signaux bruités | des **alertes peu fiables** dans le brouillard de guerre |
| Benchmark | le **score à battre** (« le marché ») |
| Long / short | parier **ça monte** / parier **ça tombe** |
| Crowding | tout le monde sur la **même file** → elle s'effondre |
| Appel de marge | **liquidation forcée** quand ta mise à levier perd trop |
| Coupons / crédit | **prêter pour un revenu régulier** — sûr, sauf si l'emprunteur fait faillite |
| Reach-for-yield | courir après le **gros revenu** = ramasser le plus risqué juste avant le krach |
| Banque centrale | le **PNJ qui contrôle la météo** ; son ton trahit le danger qui monte |

## Stratégie de communication

- **Store / trailer / première phrase** : tagline + chaises musicales. Zéro jargon. On
  vend la **tension**, pas le thème.
- **In-game** : tooltips 2 niveaux (memo §12). Le mot juste (spread, carry, drawdown) est
  là pour qui veut creuser — c'est un *easter egg de profondeur*, jamais un prérequis.
- **Règle d'or** : si une phrase de pitch contient un mot qu'un non-trader doit chercher,
  elle n'est pas pour l'extérieur. On la garde pour le niveau 2.
